#### 1、How to use

```bash
# Clone repository
git clone git@github.com:sgz886/portfolio.git

# enter project directory
cd portfolio

# install dependencies
npm install

# start project
npm start
# frontend: http://localhost:3000
# backend: http://localhost:3002

```
